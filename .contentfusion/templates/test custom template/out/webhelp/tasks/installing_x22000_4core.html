Installing X22000_4Core

You need to enter the X22000 cockpit console with admin rights.
This task explains how to install the X22000_4Core API into the X22000.
  1. Launch the console with a user having admin privileges
  2. Switch the console into maintenance mode
  3. Connect the console to the maintenance area secure network
  4. Download the core corresponding API version (from the available commercial versions repository):
    • x22000_4Core_ver1
    • x22000_4Core_ver2
    • x22000_4Core_ver3
    • x22000_4Core_ver4
  5. Once the download finishes, move onto API directory and launch the ./install command
The API is installed and appears on the bottom right of the console (in flight mode).
root@x22000:~# scp -r mirror.x22000.4core:/bin/x22000_4Core_ver3 /bin/
root@x22000:~# /bin/x22000_4Core_ver3
root@x22000:~# ./install
Once the installation finishes, reboot the cockpit console to apply the changes. After the reboot, the console will not be in maintenance mode anymore.
