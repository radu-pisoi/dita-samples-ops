Replacing Z11000 Fuel Valve

Warning
High Temperatures
Main engines combustion chamber can reach 12,000°F (6,648.9°C). Every post-mission maintenance operation should be done after 120 hours.
  1. Unscrew all the fuel valve's bolts
  2. Remove both O-rings (one at each valve end)
  3. Lubricate the new fuel valve with high-temperature grease
  4. Check both O-rings, they may need to be replaced too
  5. Mount the O-rings back on the new valve
  6. Tights the bolts using a wrench
